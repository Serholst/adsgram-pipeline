# Supply Partnership — AdsGram + Adult Ad Networks

## Идея
AdsGram становится источником inventory (Telegram) в ad networks (ExoClick, TrafficJunky).
Их рекламодатели (казино, дейтинг, adult) покупают показы в Telegram через привычный кабинет.

## Цепочка денег
```
Рекламодатель (adult) -> ExoClick/TJ (DSP/network, 30-40% cut) -> AdsGram (supply, 60-70%)
```

## Расчёт (месяц)
- AdsGram inventory для adult: ~2-3M показов/мес (~20% от 11.3M total)
- CPM на adult у ExoClick: $3-8
- Revenue gross: $6K-$24K/мес
- ExoClick cut (30-40%): $1.8K-$9.6K
- AdsGram revenue (60-70%): $4.2K-$14.4K/мес

## Выгода для ad network
- Уникальный inventory (Telegram — никто больше не предлагает)
- Adult-friendly (нет бана как в Meta/Google)
- Новые GEO с высоким CTR (Nigeria 2.5%, India 1.7%)

## Выгода для AdsGram
- Demand без своих sales — ad network приносит рекламодателей
- Масштабирование через их базу (ExoClick = 13B impressions/день)

## Inventory Exchange (доп. опция)
- Обмен трафиком: AdsGram показывает рекламу ExoClick в TG, ExoClick промоутит AdsGram на adult-сайтах
- Проблема масштаба: ExoClick 13B/день vs AdsGram ~370K/день — нужен коэффициент
- Цель для AdsGram: привлечение новых паблишеров и рекламодателей
